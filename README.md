# Bullet log

Take a log quickly like bullets.

## Usage

Add an entry to your `BULLETLOG` file on CLI

```
$ date +%Y-%m-%d
2020-05-27
$ bulletlog add Build my NES emulator
$ cat ~/.BULLETLOG
## 2020-05-27

* Build my NES emulator

## 2020-05-24

* note-cli written in Rust

* Haskell Scripting

* Try https://github.com/wellle/tmux-complete.vim
```
